IDE

  1. sublime Text
    优点：打开速度快，快捷键多，修改后执行快速
  2. webstorm
    优点：语法错误有提示，可自动保存，可在编辑器上debug，打印需要的信息

缺点：因工具包比较大，打开速度慢，修改后不会立即更新有时需要选中文件刷新，占用内存多

Debug

  1. chrome调试工具

    1)打开console查看是否有错误信息
    2)打开network查看请求
    3)打开source 添加断点，逐行执行
    4)防止跳转时看不上信息，可勾选此项打印log
    5)查看a标签上active/hover样式，选中如下:hov再勾选对应的伪类
